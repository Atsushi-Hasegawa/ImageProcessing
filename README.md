## 画像処理
-　入力された画像を合成、拡大縮小などをするリポジトリ

#### 準備
- ImageMagickをインストール
```
sudo yum -y install ImageMagick
sudo yum -y install ImageMagick-devel
```
- rmagickをインストール
```
gem install rmagick
```
- rspecをインストール
```
gem install rspec
```
- RMagickの仕様書
 - http://www.simplesystems.org/RMagick/doc/constants.html#CompositeOperator

- rekognitionをインストール
```
gem install rekognition
```
